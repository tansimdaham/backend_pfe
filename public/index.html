PharmaLearn API

Bienvenue sur l'API de PharmaLearn

Cette API fournit les services backend pour l'application PharmaLearn, une plateforme de formation pharmaceutique.

Points d'entrée API principaux :

/api - Documentation API /api/login - Authentification /api/register - Inscription

Ressources disponibles

  • Gestion des utilisateurs (administrateurs, formateurs, apprenants)
  • Gestion des cours et des quiz
  • Suivi de progression des apprenants
  • Système de messagerie
  • Gestion des réclamations
  • Génération de certificats

Test de connexion

Le serveur backend est correctement configuré et fonctionne.

Vous pouvez également tester la page test.html pour vérifier le fonctionnement des fichiers statiques.
